Educational Outcomes Markup

Schema

Note: Links in this section will work after the RDF is checked in to upload our schema to schema.googleapis.com.
  • New GoogleAPI property educationalOutcomes for schema.org type CollegeOrUniversity
  • with expected value of the new GoogleAPI type EducationalOutcomes
  • New GoogleAPI types EducationalFacet and EducationalOutcome

Template

    {
      "@context": "https://schema.org",
      "@type": "CollegeOrUniversity",
      College name, address, website, any other global identifiers.

      "educationalOutcomes": {  // New property of CollegeOrUniversity.
        An object of the new GoogleAPI type EducationalOutcomes.

        Facets to restrict the cohort to a subset of graduates from this college.
        Three optional properties:  fieldOfStudyFacet, companyFacet, and functionFacet.
        Each of type EducationalFacet, with name, identifier, optionally sameAs,
        and optionally thumbnail. These facets can be present in any combination, or none
        or all of them, depending on which page is being marked up (the college or one of the
        deeper pages for a faceted slice of the college).

        Outcomes
        Outcomes for (this slice of) college graduates.
        Three optional properties:  fieldOfStudyOutcomes, companyOutcomes, and functionOutcomes.
        Each an array of type EducationalOutcome, with all the same fields
        as EducationalFacet plus numStudents.
      }
    }
